# ✨ 関西の誇り ✨

## Required Software

- [Homebrew](https://brew.sh/index_ja)
- [Yarn](https://yarnpkg.com/)

## Setup

1. yarn install
2. http://localhost:3000/ にアクセス

## Storybook

1. yarn storybook
2. http://localhost:6006/ にアクセス

## react-icon

1. `$ yarn add yarn add react-icons --save`
2. `$ yarn add @react-icons/all-files --save`
